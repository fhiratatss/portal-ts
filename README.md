# 📱 Atalho do Web App - Portal TS Service (Fukui)

Este repositório contém as configurações e o script de redirecionamento para gerar o ícone corporativo da **TS Service** na Tela de Início de dispositivos móveis (iOS / Android).

---

## 🔗 Links Oficiais

* **Link de Distribuição (GitHub Pages):** `https://fhiratatss.github.io/portal-ts/`
* **Destino do Portal (Google Sites):** `https://sites.google.com/tsservice.ne.jp/portaltsservicefukui/home`

---

## 🛠️ Estrutura do Projeto

* `index.html`: Código responsável por carregar as metatags da logo corporativa e realizar o redirecionamento.
* `logo.png`: Imagem do ícone exibida na Tela de Início dos celulares.
* `manifest.json`: Arquivo de configuração de Web App para navegadores Android.

---

## 📲 Como orientar a instalação nos aparelhos:

### 🍎 iPhone (Safari)
1. Acesse o link de distribuição no Safari.
2. Enquanto carrega a tela inicial, clique no botão **Compartilhar** 📤.
3. Selecione **Adicionar à Tela de Início**.
4. Confirme no botão **Adicionar**.

### 🤖 Android (Chrome)
1. Acesse o link de distribuição no Chrome.
2. Toque no menu de três pontinhos `⋮` (canto superior direito).
3. Selecione **Adicionar à tela inicial** (ou *Instalar aplicativo*).
